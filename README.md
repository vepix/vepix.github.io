# Victor Dong | Mechanical Engineering Portfolio

This repository contains the source code for my personal mechanical engineering portfolio website, built as a static site and deployed with GitHub Pages.

The site highlights my experience in mechanical design, simulation, prototyping, manufacturing, and vehicle systems engineering, with dedicated project pages for selected work.

## About

This portfolio was created to showcase my background in:

- CAD and design for manufacturing
- Simulation and FEA
- Vehicle dynamics
- Prototyping and fabrication
- Research and engineering projects

The website is designed to be minimal, fast, and easy to navigate, with separate pages for major projects.
