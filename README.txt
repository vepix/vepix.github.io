# Victor Dong Portfolio Site

This folder contains a static multi-page portfolio site built for GitHub Pages.

## Files
- `index.html` — homepage
- `styles.css` — shared styles
- `projects/*.html` — dedicated project pages
- `assets/Victor_Dong_Resume.pdf` — resume download

## How to use with GitHub Pages
1. Upload all files in this folder to the root of your GitHub repository.
2. In GitHub repo settings, make sure Pages is publishing from the branch/folder where `index.html` lives.
3. Visit your GitHub Pages URL to preview the site.
4. Replace placeholder visuals with your own CAD renders, screenshots, or photos as you gather them.
