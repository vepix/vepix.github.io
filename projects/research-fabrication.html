What this project involved

At Sant’Anna University, designed a robotic probe in Fusion 360 to collect subsurface root structure data and collaborated on a 2-DOF robotic system with differential linear and rotational motion.
Designed the agricultural robot for field deployment by integrating watertight seals, data retrieval systems, and solar-powered operation.
Designed a custom FDM 3D printer with differential CoreXY motion and a 4-motor Z gantry.
Implemented automatic bed leveling using a probing routine to generate a 3D bed mesh and compensate for thermal warping.
Integrated an LCD touchscreen, MCU, and Raspberry Pi for real-time sensor visualization and remote print queue management, while using Python and IMU data to tune motor control behavior.
